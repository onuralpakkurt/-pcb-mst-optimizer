# AI Prompt Dökümleri

> Bu doküman AI servisine gönderilen tüm prompt'ları tarih + amaç + sonuç ile birlikte tutar. Şartname gereği teslim edilecek.

## Format

Her prompt için:

```
### YYYY-MM-DD — [Görev kısa adı]

**Amaç:** [Ne için kullanıldı]

**Model:** Gemini Pro / GPT-4 / vb.

**Prompt:**
\`\`\`
[Tam prompt metni]
\`\`\`

**Cevap özeti:** [Modelin cevabının özeti veya tam metni]

**Sonuç:** [Faydalı oldu mu, ne işe yaradı]
```
